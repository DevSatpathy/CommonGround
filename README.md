# CommonGround
Where2Meet, When2Meet.

# Prereqs
- Python v2.7.8+
- pip

# Instructions
1. Go the the root directory (commonground)
2. `pip install -r requirements.txt`
3. `python __init__.py`

# Actual Instructions
1. Go the the root directory (commonground)
2. `pip install -r requirements.txt`
3. `set FLASK_APP=app.py`
4. `export FLASK_APP=app.py`
5. `flask run`
6. in your browser, open localhost:5000
